# General Knowledge Mini Quiz!

---

## Description / Overview
A mini quiz game featuring a pool of general knowledge questions that are randomly selected each round. The quiz is designed to be short, interactive, and replayable, giving users a different experience every time they play.

---

## Features
- Prompts the player to enter their name before starting  
- Randomly selects questions from a larger question bank  
- Asks only 3 questions per round for a quick experience  
- Multiple-choice format with 4 possible answers per question *(GUI version)*  
- Allows a second attempt if a question is answered incorrectly  
- Tracks the player’s score throughout the quiz  
- Displays final score and feedback at the end  
- Clean and interactive interface using `tkinter` *(GUI version)*  

---

## How to Run the Quiz / Start It
1. Make sure Python is installed on your system  
2. Download or copy the quiz code  
3. Open the file in a Python-supported IDE (e.g., VS Code, IDLE)  
4. Paste the code if needed and save the file  
5. Run the program:
   - Press **F5**, or  
   - Click the **Run** button in your IDE  
6. Follow the on-screen instructions and answer the questions
